This tool is designed to simplify your keyword research process. Here are the core functionalities and metrics:

Core Functionalities

  • **Single Keyword Search:** Type a keyword and select the target **Country** (e.g., US, IN) for localized data, then hit **Search Keyword**.
  • **Bulk Upload:** Use the **Upload CSV (Bulk)** button to process lists of keywords instantly, saving significant time.
  • **Export Results:** Use the **Export CSV** button to download all current search results for further analysis or documentation.

Key Metrics Explained

  • **Volume:** The average number of monthly searches for that keyword.
  • **KD (Keyword Difficulty):** A score (0-100) indicating the competition level. **Low KD (Green) means easier ranking.**
  • **Intent:** Identifies the user's goal‚Äîis the user looking for **Information** (e.g., *how to fix a car*), making a **Transaction** (e.g., *buy cheap shoes*), or something else?
  • **CPC (Cost Per Click):** The average price advertisers pay for a single click on this keyword in Google Ads.
  • **Trend:** A visual indicator ($\text{üìà}$) showing the recent search interest trend.
  • **SERP (Search Engine Results Page):** Click the **SERP** button to see a snapshot of the top-ranking pages, including their **Domain Authority** and **On-page Score**.
